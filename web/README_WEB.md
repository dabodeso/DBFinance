# Web (Streamlit)

Este folder contiene una versión desplegable en web de la app.

## Entry point

`web/streamlit_app.py`

## Dependencias

Streamlit Community Cloud instala dependencias desde `web/requirements.txt` (prioridad sobre el `requirements.txt` de la raíz).

## Qué muestra

- Tabs: `Cuenta de resultados`, `Balance`, `Flujos de caja`
- Tabla (formateada)
- Selección de 1-2 métricas para graficar con YoY en el hover
